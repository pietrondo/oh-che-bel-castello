# 🏰 oh che bel castello

Un RTS gestionale medievale avanzato per browser, dove vesti i panni di un sovrano incaricato di trasformare un piccolo mastio in un fiorente regno. 

## 🌟 Caratteristiche Principali

### 📈 Gestione Economica e Sociale
*   **Risorse Multiple**: Gestisci Wood, Stone, Food, Gold, Iron, Tools, Knowledge e Prestige.
*   **Classi Sociali**: La tua popolazione evolve da Contadini a Cittadini e Nobili in base alla felicità e al prestigio. Ogni classe ha consumi e capacità di tassazione differenti.
*   **Forza Lavoro**: Assegna manualmente i lavoratori agli edifici per ottimizzare la produzione.

### 🗺️ Mappa procedurale
*   **Generazione Dinamica**: Ogni partita crea una mappa unica con Foreste, Miniere, Montagne invalicabili e Fiumi.
*   **Vincoli di Terreno**: Costruisci segherie e miniere solo su nodi di risorsa naturali.

### 🧪 Ricerca e Progresso
*   **Albero Tecnologico**: Costruisci l'Università per generare Conoscenza e sbloccare bonus permanenti come la Rotazione delle Colture o Tecniche Minerarie.
*   **Mercato Dinamico**: Prezzi che fluttuano ogni giorno. Compra basso e vendi alto per finanziare l'espansione del regno.

### 🎭 Narrazione e Politica
*   **Dialoghi Profondi**: Interagisci con i tuoi consiglieri attraverso un sistema di dialoghi immersivo.
*   **Dilemmi Morali**: Prendi decisioni cruciali che influenzano la felicità del popolo e le risorse del tesoro.
*   **Sistema di Difesa**: Proteggi il regno dalle incursioni nemiche costruendo caserme e mantenendo un alto rating di difesa.

### 💾 Persistenza
*   **Salvataggio Automatico**: I tuoi progressi vengono salvati localmente nel browser (localStorage).

## 🛠️ Tecnologie Utilizzate
*   **React** (v18)
*   **TypeScript**
*   **Vite**
*   **Vanilla CSS** per un'estetica medievale pulita e performante.

## 🚀 Come Iniziare

1.  **Clona il repository**:
    ```bash
    git clone git@github.com:pietrondo/oh-che-bel-castello.git
    cd oh-che-bel-castello
    ```

2.  **Installa le dipendenze**:
    ```bash
    npm install
    ```

3.  **Avvia il server di sviluppo**:
    ```bash
    npm run dev
    ```

4.  Apri l'URL fornito (solitamente `http://localhost:5173`) nel tuo browser.

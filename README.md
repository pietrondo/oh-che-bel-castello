# 🏰 oh che bel castello

Un RTS gestionale medievale avanzato per browser, dove vesti i panni di un sovrano incaricato di trasformare un piccolo mastio in un fiorente impero.

## 🌟 Caratteristiche Avanzate

### 💎 Design System Premium
*   **Interfaccia PC-Like**: Navigazione verticale a sinistra per un'esperienza desktop immersiva.
*   **HUD Ottimizzato**: Badge dinamici con feedback cromatico per le risorse critiche.
*   **Architettura Organizzata**: Sidebar costruzioni divisa in categorie (Infrastrutture, Estrazione, Produzione, Istituzioni).

### 📈 Simulazione Economica e Sociale
*   **Filiere di Consumo**: Gestisci risorse di lusso come Birra, Vestiti e Gioielli per soddisfare Cittadini e Nobili.
*   **Sistema Finanziario**: Monitora Inflazione e Debito. L'eccesso di oro non investito genera svalutazione.
*   **Politica delle Gilde**: Media tra Mercanti, Clero e Militari. Ogni tua legge sposta il favore delle fazioni.

### 🗺️ Urbanistica e Logistica
*   **Efficienza Stradale**: Costruisci strade (🛣️) per potenziare la produzione degli edifici vicini del 20%.
*   **Salute Pubblica**: Previeni le epidemie costruendo Pozzi (⛲). La densità senza igiene decima la popolazione.
*   **Meteo Dinamico**: Adattati a Sole, Pioggia, Tempesta e Neve. L'inverno richiede legno per il riscaldamento.

### 📜 Diplomazia e Leggi
*   **Mappa Mondo**: Interagisci con regni confinanti (Wessex, Mercia, Vichinghi) e gestisci le relazioni diplomatiche.
*   **Editti Regi**: Proclama leggi (Lavoro Forzato, Decima, Leva) per plasmare il futuro del tuo regno.

## 🛠️ Tecnologie
*   **React 18** + **TypeScript**
*   **Vite** (Build Tool)
*   **Vanilla CSS** (Custom Design System con variabili dinamiche)

## 🚀 Installazione Rapida

1.  `git clone git@github.com:pietrondo/oh-che-bel-castello.git`
2.  `cd oh-che-bel-castello`
3.  `npm install`
4.  `npm run dev`
